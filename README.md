# Launching

## Before start 

Install packages 

### npm i

Launch server 

### node server

Then you are ready to run app

## Available Scripts

To run app use 

### `npm start`


To run tests use

### `npm test`

To build app use

### `npm run build`


## If tests crushing or doesn't work use 

### npm i -D --exact jest-watch-typeahead@0.6.5
